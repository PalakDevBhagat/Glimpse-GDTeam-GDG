# Glimpse-GDTeam-GDG
Glimpse is a product concept that simplifies Google updates through intuitive UI/UX, clear content, and engaging visual design. This repository contains branding assets, mobile UI screens, motion graphics, Instagram carousel campaigns, mockups, and other design materials developed for the project.
